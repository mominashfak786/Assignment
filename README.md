<h2>How to Run the Project</h2>

<ol>
  <li>Clone the project repository from <a href="https://github.com/mominashfak786/Assignment.git">GitHub</a>.</li>
  <li>Open the project in Visual Studio Code.</li>
  <li>Run the <code>WeatherApp.java</code> file.</li>
  <li>The app will establish a connection to the OpenWeatherMap API and fetch hourly weather forecast data for London.</li>
  <li>The user will be prompted to choose an option:</li>
</ol>

<ul>
  <li>Enter <p>1</p> to get temperature for a specific date and time.</li>
  <li>Enter <p>2</p> to get wind speed for a specific date and time.</li>
  <li>Enter <p>3</p> to get pressure for a specific date and time.</li>
  <li>Enter <p>0</p> to exit the app.</li>
</ul>

<p>Based on the user's input, the app will display the requested weather information.</p>
